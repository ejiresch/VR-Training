Shortcuts
P...Finished direkt einen Task in einem Prozess
L...Developer Mode wird aktiviert, 3 Buttons mit Einstellungen sind dann an der Wand

Device Sim
Wenn man den Device Simulator verwenden will, muss man um ihn ein/auszuschalten jeweils XR Origin (XR Rig) 
und XR Device Simulator aktivieren/deaktivieren.
Mit WASD kann man sich im Raum bewegen und mit Q und E kann man seine höhe verändern. Mit Tab kann man die linke
Hand drehen und mit Z die rechte Hand. Mit Linksklick kann man Buttons usw. betätigen. Wenn man eine Hand mit Tab
oder Z ausgewählt hat, kann man mit R die Hand an sich bewegen.
Grundsätzlich eignet sich der Device Sim nicht zum kompletten durchspielen von einem Task sondern mehr zum Testen ob z.B.
Buttons funktionieren oder Gegenstände funktionieren wie sie sollten. 
Auch wichtig ist dass Dinge wie Animationen immer auch auf das Xr Rig (Xr Origin(XR Rig)) angewendet werden müssen da es
sozusagen zwei getrennte Skelette gibt. Das zurzeit verwendete Rig für die VR Brille ist schon depricated, dass heißt wenn
man mal updated sollte man das Device Sim Rig verwenden können.


Autostart von Tasks
Wenn man einen Autostart von einem neuen Prozess in einem ProcessScriptableObject setzen will,
muss man vor der Nummer 10 stehen haben. z.B. Für Task 1 -> 101, Task 2 -> 202...
